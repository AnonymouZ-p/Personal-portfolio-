Introduction

Python is a high-level, interpreted programming language.

It is often used for web development, scientific computing, and data analysis.

print("Hello, World!")
Variables and Data Types

In Python, you don't need to declare the data type of a variable.

The data type is determined by the assigned value.

x = 5 y = "hello"
  • Integer
  • Float
  • String
  • Boolean
  • Lists
  • Dictionaries
  • Tuples
  • Sets
Control Structures

Python has several control structures, including if-else statements and loops.

if x > 5:
print("x is greater than 5")

Loops can be used to iterate over sequences.

for i in range(5):
print(i)

Another common loop is the while loop:

count = 0
while count < 5:
print(count)
count += 1
Functions

Functions are reusable blocks of code.

def greet(name):
print("Hello, " + name + "!")

Functions can take arguments and return values.

def add(a, b):
return a + b
result = add(3, 5)
print(result) # Output: 8
Modules

Modules are pre-written code that can be imported into your program to extend its functionality.

import math

Modules can be used to perform specific tasks, such as mathematical operations.

print(math.pi)

You can also import specific functions or variables from a module:

from math import sqrt
print(sqrt(16)) # Output: 4.0
